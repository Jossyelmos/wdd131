Health Habit Tracker

Site Name

Health Habits Tracker

The name of this website conveys the function of the site. This is a website that helps users track their health habits.

Site Purpose

  • The website is designed to help individuals build and maintain healthier lifestyle by monitoring their daily habits.
  • This site allows users to track key wellness activities such as water intake, exercise, sleep, mindfulness, and balance meals.
  • Users can developconsistency, stay accountable, and work toward long-term health goals by recording their progress and receiving motivational feedback.

Scenerio

  • How can I keep track of my daily health habits, like water intake and exercise, in one place?
  • Can I save my daily progress so I can review it later?
  • Does the tracker allow me to customize or add my own habits to monitor?
  • Where can I find simple tips and tools to help me stay consistent with my health goals?

Color Schema

  • Green - This colour represent good health and wealth, tranquility and nature. This color will be used as background as it will dominate and represent the website identity.
  • Red - Represent passion, energy, urgency and excitement. This will be used for buttons on the website
  • Gray - Represent neutral, simplicity, and calm. This will be used for borders, text-color.

Typography

  • Acme - Headings(H1, h2 ....)
  • Roboto, sans-serif - Body(paragraph, links, footer, main)
